# Мобильный банковский клиент

Демонстрационное Flutter-приложение мобильного банка с синтетическими данными, русскоязычным интерфейсом, локальным хранением настроек и современной UI-структурой.

## Где разработан проект

Проект разрабатывался и редактировался в `Android Studio` как Flutter-проект на Dart.

## Что есть в проекте

- главный экран с картами, быстрыми действиями, финансовым обзором и историей операций
- боковое меню с разделами по жизненным ситуациям
- экран профиля с локальной авторизацией по номеру телефона
- экран уведомлений
- светлая и темная тема
- выбор цветовой палитры и акцентного цвета
- родительский контроль
- синтетические сценарии пополнения, оплаты и перевода

## Что нужно для запуска на другом устройстве

Перед запуском на другом компьютере или устройстве должно быть установлено:

- `Flutter SDK`
- `Android Studio`
- `Android SDK`
- эмулятор Android или подключенное физическое Android-устройство

Также желательно проверить, что команда `flutter --version` выполняется в терминале без ошибок.

## Порядок запуска проекта

1. Откройте проект в `Android Studio`.

2. Откройте встроенный терминал в корне проекта.

3. Установите зависимости:

```bash
flutter pub get
```

4. Если проект запускается на новой машине и отсутствуют платформенные служебные файлы, выполните:

```bash
flutter create .
```

5. Проверьте, что Flutter видит доступные устройства:

```bash
flutter devices
```

6. Запустите приложение на выбранном устройстве:

```bash
flutter run
```

## Запуск через Android Studio

Можно запустить проект без ручной команды `flutter run`:

1. Откройте `Android Studio`
2. Дождитесь загрузки зависимостей
3. Выберите эмулятор или подключенное устройство в верхней панели
4. Нажмите `Run`

## Полезные команды

Получение зависимостей:

```bash
flutter pub get
```

Проверка проекта:

```bash
flutter analyze
```

Запуск тестов:

```bash
flutter test
```

Сборка debug-версии:

```bash
flutter build apk --debug
```

## Примечания

- приложение не использует реальный backend
- все данные в проекте синтетические или локально сохраненные
- все пользовательские тексты отображаются на русском языке
- настройки темы, палитры, профиля и упрощенного режима сохраняются локально
